# Humble Oak Wellness

Pilot website for **Humble Oak Wellness** — an allied health clinic in Abbotsford, BC (Fraser Valley). Part of the Humble Oak family, offering registered massage therapy and counselling.

Built with the same static-site approach as [Humble Oak Solutions](https://www.humbleoaksolutions.com): HTML, CSS, and vanilla JS — no build step.

## Pages

- **Home** — `/`
- **About** — `/about-us/`
- **Contact** — `/contact-us/`

## Stakeholder Preview

This is a pilot for internal review. Content marked as placeholder; site is set to `noindex`.

**Live preview (GitHub Pages):** https://humbleoakadmin.github.io/Humble-Oak-Wellness/

## Domain & Email

**humbleoakwellness.com** is registered on IONOS and used for email. DNS is intentionally unchanged — the website is hosted on GitHub Pages only until a future launch decision.
